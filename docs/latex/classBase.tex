\doxysubsubsection{\texorpdfstring{percentile()}{percentile()}}
{\footnotesize\ttfamily Base\+::percentile (\begin{DoxyParamCaption}\item[{}]{\$p }\end{DoxyParamCaption})}

The pth percentile is the value such that p\% of the a sorted data set is smaller than it, and (100 -\/ p)\% of the data is larger.

A quick algorithm to pick the appropriate value from a sorted data set is as follows\+:


\begin{DoxyItemize}
\item Count the number of values\+: n
\item Calculate the position of the value in the data list\+: i = p $\ast$ (n + 1)
\item if i is an integer, return the data at that position
\item if i $<$ 1, return the minimum of the data set
\item if i $>$ n, return the maximum of the data set
\item otherwise, average the entries at adjacent positions to i
\end{DoxyItemize}

The median is the 50th percentile value.
