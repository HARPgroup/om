Connect to the database and create it if it doesn\textquotesingle{}t exist

Don\textquotesingle{}t call this method directly. Use \mbox{\hyperlink{classDB_aeb85aee47641c578522883780a06f7e2}{D\+B\+::connect()}} instead.

\mbox{\hyperlink{classPEAR}{P\+E\+AR}} \mbox{\hyperlink{classDB}{DB}}\textquotesingle{}s dbase driver supports the following extra D\+SN options\+:
\begin{DoxyItemize}
\item mode An integer specifying the read/write mode to use (0 = read only, 1 = write only, 2 = read/write). Available since \mbox{\hyperlink{classPEAR}{P\+E\+AR}} \mbox{\hyperlink{classDB}{DB}} 1.\+7.\+0.
\item fields An array of arrays that \mbox{\hyperlink{categoryPHP}{P\+HP}}\textquotesingle{}s dbase\+\_\+create() function needs to create a new database. This information is used if the d\+Base file specified in the \char`\"{}database\char`\"{} segment of the D\+SN does not exist. For more info, see the \mbox{\hyperlink{categoryPHP}{P\+HP}} manual\textquotesingle{}s \mbox{\hyperlink{}{dbase\+\_\+create()}} page. Available since \mbox{\hyperlink{classPEAR}{P\+E\+AR}} \mbox{\hyperlink{classDB}{DB}} 1.\+7.\+0.
\end{DoxyItemize}

Example of how to connect and establish a new d\+Base file if necessary\+: {\ttfamily  require\+\_\+once \textquotesingle{}D\+B.\+php\textquotesingle{};}

{\ttfamily \$dsn = array( \textquotesingle{}phptype\textquotesingle{} =$>$ \textquotesingle{}dbase\textquotesingle{}, \textquotesingle{}database\textquotesingle{} =$>$ \textquotesingle{}/path/and/name/of/dbase/file\textquotesingle{}, \textquotesingle{}mode\textquotesingle{} =$>$ 2, \textquotesingle{}fields\textquotesingle{} =$>$ array( array(\textquotesingle{}a\textquotesingle{}, \textquotesingle{}N\textquotesingle{}, 5, 0), array(\textquotesingle{}b\textquotesingle{}, \textquotesingle{}C\textquotesingle{}, 40), array(\textquotesingle{}c\textquotesingle{}, \textquotesingle{}C\textquotesingle{}, 255), array(\textquotesingle{}d\textquotesingle{}, \textquotesingle{}C\textquotesingle{}, 20), ), ); \$options = array( \textquotesingle{}debug\textquotesingle{} =$>$ 2, \textquotesingle{}portability\textquotesingle{} =$>$ D\+B\+\_\+\+P\+O\+R\+T\+A\+B\+I\+L\+I\+T\+Y\+\_\+\+A\+LL, );}
