Variable GlossaryShow/Hide:

Out-flow Variables:
  • Qreach - The value in cfs of the flow at the outlet of the modeling segment that contains the control point (segment boundaries shown in the map labelled "Select Best Model Location" above). This value will reflect the cumulative outflows of the containing model segment after considering all withdrawals, discharges, and tributary inflows.
  • Qintake - The mean daily flow value in cfs at the intake for this withdrawal (for "Off-Stream Storage" or "Direct Withdrawal" projects only).
Runoff/Inflow Variables:
  • Runit - Unit area inflow of runoff, baseflow and interflow to the containing model segment in cfs/sqmi.
  • imp_inflow - Inflow to the impoundment in cfs.
  • surrogate_gage_Qout - Surrogate gage flow in cfs (surrogate gage ID specified above in "Advanced Settings")
  • surrogate_gage_Qunit - Surrogate gage flow in cfs/square mile.
  • surrogate_gage_area - Surrogate gage drainage area in square miles.
  • drainage_area - Impoundment drainage area in square miles.
  • month - The numerical index of the current calendar month in the modeling simulation (1-12).
Withdrawal Amount Variables:
  • wd_mgd - Withdrawal value simulated in this segment based on the selected model inputs.
  • wd_upstream_mgd - Cumulative withdrawals in all model segments flowing into this model segment.
  • safeyield_mgd (Local Only) - Daily rate of withdrawal in this segment according to the withdrawal info entered in the section "Custom Withdrawal" (MGD).
  • wd_vahydro_mgd (VAHydro Scenario) - Daily rate of withdrawals in this segment according to the selected scenario of the containing VAHydro model.
  • wd_vwuds_mgd (VWUDS Historic) - Historic daily withdrawal rate in this segment as reported under Virginia Water Use Data reporting program (MGD).
  • wd_wsp_current_mgy (WSP Current) - Current annual withdrawal in this segment as reported under water supply planning regulation (MGY).
  • wd_wsp_current_mgd (WSP Current) - Current daily withdrawal rate in this segment as reported under water supply planning regulation (MGD).
  • wd_wsp_future_mgd (WSP 2040) - Year 2040 projected daily withdrawal rate in this segment as reported under water supply planning regulation (MGD).
  • wd_wsp_max_mgd (WSP 2040) - Maximum allowable daily withdrawal rate in this segment as reported under water supply planning regulation (MGD). wd_wsp_current_gw_mgd wd_wsp_max_gw_mgd wd_wsp_future_gw_mgd
Impoundment Information Variables:
  • lake_elev - The current depth of water in the parent container impoundment.
  • impoundment_lake_elev - The current depth of water in the impoundment located in this control point.
  • simple_release - The value calculated in the equation above labelled "Simple Release".
  • tiered_release - The value of the calculation yielded by the rules in the section labelled "Tiered Release".
Withdrawal/Release Rule Variables:
  • tiered_flowby - The value of the calculation yielded by the rules in the section labelled "Tiered Flow-By".
  • simple_flowby - The value calculated in the equation above labelled "Simple Flow-By".
  • simple_release - The value calculated in the equation above labelled "Simple Release".
  • tiered_release - The value of the calculation yielded by the rules in the section labelled "Tiered Release".
VAHydro Model Run Descriptions:
  • Run 0 - "Baseline" scenario, no withdrawals, no point sources, pre-development land use.
  • Run 1 - Historical scenario, VWUDS reported withdrawals, VPDES discharges, land use varying over 1984-2005 according to CBP Phase 5 model.
  • Run 2 - "Current" scenario, mean VWUDS withdrawals & VPDES discharges from 2005-2009, 2005 land use.
  • Run 3 - Future, full build out scenario.
  • Run 4 - "End of Permit Period" scenario - maximum permitted withdrawals for permit period ending in 2028.
