◆ prepare()

DB_ibase::prepare (   $query)

Prepares a query for multiple execution with execute().

prepare() requires a generic query as string like INSERT INTO numbers VALUES (?, ?, ?) . The ? characters are placeholders.

Three types of placeholders can be used:

  • ? a quoted scalar value, i.e. strings, integers
  • ! value is inserted 'as is'
  • & requires a file name. The file's contents get inserted into the query (i.e. saving binary data in a db)

Use backslashes to escape placeholder characters if you don't want them to be interpreted as placeholders. Example: "UPDATE foo SET col=? WHERE col='over \& under'"

Parameters
string$queryquery to be prepared
Returns
mixed DB statement resource on success. DB_Error on failure.
