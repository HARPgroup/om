error class, replaces \mbox{\hyperlink{classPEAR__Error}{P\+E\+A\+R\+\_\+\+Error}}

An instance of this class will be returned if an error occurs inside \mbox{\hyperlink{classXML__Parser}{X\+M\+L\+\_\+\+Parser}}.

There are three advantages over using the standard \mbox{\hyperlink{classPEAR__Error}{P\+E\+A\+R\+\_\+\+Error}}\+:
\begin{DoxyItemize}
\item All messages will be prefixed
\item check for \mbox{\hyperlink{classXML__Parser}{X\+M\+L\+\_\+\+Parser}} error, using is\+\_\+a( \$error, \textquotesingle{}\mbox{\hyperlink{classXML__Parser__Error}{X\+M\+L\+\_\+\+Parser\+\_\+\+Error}}\textquotesingle{} )
\item messages can be generated from the xml\+\_\+parser resource
\end{DoxyItemize}
