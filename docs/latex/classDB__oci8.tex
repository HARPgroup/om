\end{DoxyItemize}

\mbox{\hyperlink{classPEAR}{P\+E\+AR}} \mbox{\hyperlink{classDB}{DB}}\textquotesingle{}s oci8 driver supports the following extra D\+SN options\+:
\begin{DoxyItemize}

\item new\+\_\+link If set to true, causes subsequent calls to \mbox{\hyperlink{classDB__oci8_a8d984712e4c1d1b3a3fc602a3b7fe6f8}{connect()}} to return a new connection link instead of the existing one. Prepares a query for multiple execution with \mbox{\hyperlink{classDB__oci8_ae6236fd0543a79214cc39f60097254d5}{execute()}}.

With oci8, this is emulated.

\mbox{\hyperlink{classDB__oci8_a69c388117ea5123714feeda17c30faec}{prepare()}} requires a generic query as string like {\ttfamily  I\+N\+S\+E\+RT I\+N\+TO numbers V\+A\+L\+U\+ES (?, ?, ?) }. The {\ttfamily ?} characters are placeholders.

Three types of placeholders can be used\+:
\begin{DoxyItemize}
\item {\ttfamily ?} a quoted scalar value, i.\+e. strings, integers
\item {\ttfamily !} value is inserted \textquotesingle{}as is\textquotesingle{}
\item {\ttfamily \&} requires a file name. The file\textquotesingle{}s contents get inserted into the query (i.\+e. saving binary data in a db)
\end{DoxyItemize}

Use backslashes to escape placeholder characters if you don\textquotesingle{}t want them to be interpreted as placeholders. Example\+: {\ttfamily  \char`\"{}\+U\+P\+D\+A\+T\+E foo S\+E\+T col=? W\+H\+E\+R\+E col=\textquotesingle{}over \textbackslash{}\& under\textquotesingle{}\char`\"{} }
