\doxysubsection{Detailed Description}
Dependency check for \mbox{\hyperlink{classPEAR}{P\+E\+AR}} packages

The class is based on the dependency R\+FC that can be found at \href{http://cvs.php.net/cvs.php/pearweb/rfc}{\texttt{ http\+://cvs.\+php.\+net/cvs.\+php/pearweb/rfc}}. It requires \mbox{\hyperlink{categoryPHP}{P\+HP}} $>$= 4.\+1
