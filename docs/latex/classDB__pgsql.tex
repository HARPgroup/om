Connect to the database server, log in and open the database

Don\textquotesingle{}t call this method directly. Use \mbox{\hyperlink{classDB_aeb85aee47641c578522883780a06f7e2}{D\+B\+::connect()}} instead.

\mbox{\hyperlink{classPEAR}{P\+E\+AR}} \mbox{\hyperlink{classDB}{DB}}\textquotesingle{}s pgsql driver supports the following extra D\+SN options\+:
\begin{DoxyItemize}
\item connect\+\_\+timeout How many seconds to wait for a connection to be established. Available since \mbox{\hyperlink{classPEAR}{P\+E\+AR}} \mbox{\hyperlink{classDB}{DB}} 1.\+7.\+0.
\item new\+\_\+link If set to true, causes subsequent calls to \mbox{\hyperlink{classDB__pgsql_a83d5a6d5c12435f6ae996ca9cca94e7e}{connect()}} to return a new connection link instead of the existing one. W\+A\+R\+N\+I\+NG\+: this is not portable to other D\+B\+MS\textquotesingle{}s. Available only if \mbox{\hyperlink{categoryPHP}{P\+HP}} is $>$= 4.\+3.\+0 and \mbox{\hyperlink{classPEAR}{P\+E\+AR}} \mbox{\hyperlink{classDB}{DB}} is $>$= 1.\+7.\+0.
\item options Command line options to be sent to the server. Available since \mbox{\hyperlink{classPEAR}{P\+E\+AR}} \mbox{\hyperlink{classDB}{DB}} 1.\+6.\+4.
\item service Specifies a service name in pg\+\_\+service.\+conf that holds additional connection parameters. Available since \mbox{\hyperlink{classPEAR}{P\+E\+AR}} \mbox{\hyperlink{classDB}{DB}} 1.\+7.\+0.
\item sslmode How should S\+SL be used when connecting? Values\+: disable, allow, prefer or require. Available since \mbox{\hyperlink{classPEAR}{P\+E\+AR}} \mbox{\hyperlink{classDB}{DB}} 1.\+7.\+0.
\item tty This was used to specify where to send server debug output. Available since \mbox{\hyperlink{classPEAR}{P\+E\+AR}} \mbox{\hyperlink{classDB}{DB}} 1.\+6.\+4.
\end{DoxyItemize}
